# Molchip Software AI · 工作台

一个现代化的项目管理和路线图展示系统，专为团队协作和任务跟踪而设计。

## ✨ 功能特点

### 核心功能
- **双视图模式**：架构视图和负责人视图，满足不同的管理需求
- **项目管理**：创建、编辑、删除项目，支持分类管理
- **进度跟踪**：时间线式的里程碑记录，实时编辑项目进度
- **状态管理**：待开发/已完成状态切换

### 高优先级任务系统 🔥
- **唯一约束**：每个负责人只能设置一个高优先级任务
- **智能管理**：设置新的高优先级任务时自动取消之前的设置
- **视觉突出**：紫色边框和专门的标识系统
- **权限控制**：只有项目负责人才能设置高优先级

### 筛选与搜索
- **待开发筛选**：快速查看未完成的任务
- **高优先级筛选**：专注于重要任务
- **组合筛选**：支持多种筛选条件同时使用

### 编辑功能
- **高级编辑模式**：提供完整的项目管理功能
- **内联编辑**：双击即可编辑项目信息
- **批量操作**：支持负责人批量编辑
- **GitHub集成**：项目可关联GitHub仓库

### 用户体验
- **响应式设计**：完美适配桌面端和移动端
- **瀑布流布局**：动态调整卡片排列
- **实时反馈**：操作成功提示和错误处理
- **权限控制**：基于登录状态的功能访问控制

## 🛠️ 技术栈

### 前端
- **HTML5/CSS3**：现代化的语义化标记和样式
- **Vanilla JavaScript**：原生JS实现，无框架依赖
- **响应式设计**：CSS媒体查询适配多种设备

### 后端
- **Supabase**：现代化的Firebase替代方案
  - 实时数据库
  - 用户认证
  - 行级安全(RLS)策略

### 设计系统
- **CSS变量**：统一的颜色和间距系统
- **组件化样式**：模块化的CSS架构
- **主题色彩**：
  - 黄色：待开发状态
  - 紫色：高优先级任务
  - 绿色：GitHub集成
  - 灰色：编辑模式

## 🚀 快速开始

### 本地开发
```bash
# 克隆项目
git clone https://github.com/molchip-tpr/molchip_ai_roadmap.git

# 进入项目目录
cd molchip_ai_roadmap

# 启动本地服务器
python3 -m http.server 8000

# 访问应用
open http://localhost:8000
```

### 配置要求
- 需要配置Supabase项目
- 设置相应的数据库表结构
- 配置RLS（行级安全）策略
